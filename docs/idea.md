# The Black Cat Brew

"The Black Cat Brew" es una aplicación web inspirada en la estética del café y los tonos oscuros, haciendo referencia a bebidas intensas como el cold brew y a la elegancia asociada a los gatos negros. El nombre aporta identidad visual y personalidad al proyecto.


## Descripción general

The Black Cat Brew es una aplicación web fullstack que permite a los usuarios explorar distintos tipos de café según su origen y características, así como guardar sus cafés favoritos para consultarlos posteriormente, ademas de poder tomar notas y calificarlos.

La aplicación combina una interfaz moderna con un backend que gestiona los datos de los cafés mediante una API REST.

## Problema que resuelve

Muchas personas consumen café pero no conocen bien las diferencias entre los distintos tipos disponibles en el mercado. Conceptos como intensidad, acidez o notas de sabor pueden resultar confusos.

Esta aplicación ayuda a:

- Entender mejor las características del café
- Descubrir nuevos tipos según preferencias
- Tener una referencia visual antes de comprar


## Usuario objetivo

La aplicación está dirigida a:

- Personas interesadas en el mundo del café
- Usuarios que quieren aprender sobre distintos tipos de café
- Personas que buscan recomendaciones antes de comprar
- Consumidores habituales que quieren organizar sus cafés favoritos


## Funcionalidades principales

- Visualizar un catálogo de cafés
- Mostrar información detallada de cada café:
  - origen
  - intensidad
  - acidez
  - descripción
  - imagen
- Acceder a enlaces externos de compra 
- Añadir cafés a una lista de favoritos
- Rellenar valoración en tabla
- Eliminar cafés de favoritos

## Funcionalidades opcionales (mejoras futuras)

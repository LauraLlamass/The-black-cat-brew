# Componentes

## CoffeeCard

Componente reutilizable que muestra la información de un café.

### Props
- coffee: objeto tipado con información del café (id, name, origin, etc.)

### Funcionalidad
- Muestra nombre, origen, descripción e imagen
- Permite añadir el café a favoritos mediante un botón

### Estilos
- Diseñado con Tailwind CSS
- Uso de bordes redondeados y paleta de color café

## Navbar

Barra de navegación principal de la aplicación.

### Funcionalidad
- Navegación entre páginas: Home, Catálogo y Favoritos
- Uso de React Router (`Link`)

## Hero

Sección principal de la página de inicio.

### Funcionalidad
- Presenta la aplicación
- Contiene título, texto descriptivo e imagen

## InfoStrip

Sección informativa que describe el propósito del proyecto.

### Funcionalidad
- Explica la idea de la aplicación
- Incluye imagen decorativa

## SectionHeading

Componente reutilizable para títulos de sección.

### Props
- eyebrow: texto pequeño superior
- title: título principal
- description: texto descriptivo

## Footer

Pie de página de la aplicación.

### Funcionalidad
- Información básica del proyecto
- Cierre visual de la página
